The Odin Project

The Odin Project mission

At the Odin Project you can learn different types of languages like:

  • Rubi
  • Java script
  • HTML
  • CSS

The goal of The Odin Project is to provide a complete path for students to go from zero programming knowledge to employed as web developers while working with other students and picking up the skills to become self sufficient along the way.

You can read extra information at the odin project to learn even more about the values and principles that guide the pursuit of our mission.
